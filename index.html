📅 Agendamento — Google Agenda
🔧 Como obter as credenciais do Google
Passo 1: Acesse console.cloud.google.com → Criar projeto → Ativar API do Google Calendar
Passo 2: APIs e Serviços → Credenciais → Criar credenciais → ID de cliente OAuth
   Tipo: Aplicativo da Web → URI de redirecionamento autorizado:
   https://developers.google.com/oauthplayground
Passo 3: Anote o Client ID e o Client Secret
Passo 4: Acesse developers.google.com/oauthplayground
   → Clique no ícone ⚙ → marque "Use your own OAuth credentials" → cole Client ID e Secret
   → Em "Step 1" selecione Google Calendar API v3 → escopo calendar
   → Authorize → "Step 2" → Exchange authorization code → copie o Refresh Token
